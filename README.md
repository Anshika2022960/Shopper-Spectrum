# 🛒 Shopper Spectrum

## Customer Segmentation and Product Recommendation in E-Commerce

---

## 📌 Project Overview

Shopper Spectrum is an end-to-end Machine Learning project that helps e-commerce businesses understand customer purchasing behavior and recommend similar products.

The project uses:

- RFM Analysis
- K-Means Clustering
- Elbow Method
- Silhouette Score
- Item-Based Collaborative Filtering
- Cosine Similarity
- Streamlit Dashboard

---

## 📊 Features

### 🛍 Product Recommendation

- Select or enter a product name
- Recommend Top 5 similar products
- Uses Cosine Similarity

### 👥 Customer Segmentation

Predict customer category using:

- Recency
- Frequency
- Monetary Value

Customer Segments:

- High-Value
- Regular
- Occasional
- At-Risk

### 📈 Business Insights

- Customer Segmentation Summary
- Recommended Business Actions
- ML Model Overview
